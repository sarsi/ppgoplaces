SHOPPING LIST

Shopping list provides access to add shopping item, tag to vacation, add target price, add picture and quick foreign exchange conversion text box.

Add New button
Click button to add item to shopping item.

Text box at top of screen, provides quick foreign currency conversion computation, a convenient comparison at a glance to the shopping item target price. User need to first setup foreign exchange rates of visiting country in Setup Fx Rate menu.

Item Context Menu
Long press on checklist item to access to the below:

  • Edit Item
    • Edit details of item, e.g. item description or tagging to a different category.
  • Delete Item
    • Delete item from list.

Holiday Drop Down List
Click on list to display a drop down list to display a list of vacations. Shopping list can be filtered and display Holiday that it has been tagged to.

Show/Hide Toggle Button
When Hide selected, all checked items or completed tasks will be hidden.
