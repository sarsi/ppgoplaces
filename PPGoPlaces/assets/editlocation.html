Edit Existing Location
If the location information is available and an address has not been added, The program will attempt to discover any known address if an data network is available. Use can also choose to use GoogleTM map feature to find location address.

Location Date button
To change date, tap on the date item to access a date picker dialog.

Tag Location
Tap on the drop down menu to access list of holiday or tag to Home. The location date need to be within the period of the holiday or else the system will return an error when trying to update location.

Location Name
This field cannot be left empty else the system will return an error when updating location.

Location Address
If no address is empty, when a known address is found, the program will display a Address button. User can pick the found address by clicking on the button or manually input the address if the found address is incorrect. .

MAP
User can also choose to find location and address information via the Map button, access GoogleTM mapping feature. Enter a simple location name or address, e.g. Big Ben, London, UK and select Find Address to find location. When a location is found, a red round marker will indicate the found location on the map. Select Use Location to retreive information into the Add Location screen. Note: The blue flashing dot indicates the current location of the device.
