Add New Location
This is location aware program, when a location is detected the co-ordinates will be displayed. If a data network connection is available, the program will attempt to discover the best known address of the location. Location can be discovered via GPS or network, the accuracy and discovery duration of the location and address may varies.

Location Date button
To change date, tap on the date item to access a date picker dialog.

Tag Location
Tap on the orange drop down menu to access a list of vacations, no menu button will not function if user have not create any vacation. An error will return on saving location if location date is not within the period of the vacation selected.

Location Name
A location name cannot be left empty, an error will return on saving location if the field is left blank.

Location Address
When a known address is found, the program will display a Address button. User can pick the found address by clicking on the button or manually input the address if the found address is incorrect.

Indoor/Outdoor
Select indoor to speed up location search without GPS. GPS only function operate outdoor, turn off GPS to preserve battery life. When outdoor with clear view of sky, turn on GPS and set to outdoor to get a more accurate location fix.

Stop Location Updates/Resume Location Updates
Turn off location updates to preserve battery life when location search is no longer required. Note: Location updates will stop when user leave current screen.

MAP
User can also choose to find location and address information via the Map button, access GoogleTM mapping feature. Enter a simple location name or address, e.g. Big Ben, London, UK and select Find Address to find location. When a location is found, a red round marker will indicate the found location on the map. Select Use Location to retreive information into the Add Location screen. Note: The blue flashing dot indicates the current location of the device.

Additional Information
Satellite view of the map can be selected via Settings menu, under Find Location mode
