MY PREFERENCES

Settings menu provides numerous important settings and important utilities backup and restore programs for the application main database.

  • List - Scrolling Text Enable
  • Map - Satellite View Enable
  • Find Location - Accuracy High/Slow
  • Backup Data
  • Restore Data
  • Email Signature

List - Scrolling Text Enable
When enable, location or holiday name that does not fix the row of the list view, will automatically scroll horizontally so as the user can view the full length of the text. When disable, long name display may appears as truncated.

Map - Satellite View Enable
When enable, the MAP feature will display in satellite view. When disable the map will default to street view.

Find Location - Accuracy High/Slow
When enable, the find location in Add Location will set criteria to accuracy fine, and when outdoor mode is selected it will default to finding a location fix via device GPS function. When not disable, the find location is set to Accuracy Low/Fast criteria accuracy coarse, and default to using cell tower and wifi networks to get a location fix.

Backup Data
This function will backup the user database to the application internal directory. It is a good house keeping to do frequent backup depends on the application usage. Do not attempt to backup corrupted data, as it wll overwrite the good copy of last backup copy.

Restore Data
This function will restore the user database to the backup previously make from the application internal directory. Note: Restore will overwrite data current stored in program with the older backup, depends on when the last backup, newer data created between the last backup up to the point of restoration will be lost.

Email Signature
Set user email signature here, the signature is used with Share via an email client in item menu throughout the application. When no email signature is setup, the signature will default to [sent from PPGoPlaces].
