VIEW VACATIONS

View list of vacations or holidays. Click on any holiday to display a short description of the holiday at the top of the list. If there are pictures available in any of the location(s) tagged to the selected holiday, a holiday photo album button will display on the top right of the display.

Add Holiday button
Click button to add new vacation, holiday or trip to the list.

Holiday Context Menu
Long press on holiday item to access to the below:

  • Edit
  • Delete
  • Add New Location
  • View Locations
  • View Locations Map
  • View HOME Locations

Edit
Edit holiday details.

Delete
Delete selected holiday. Locations previously tagged to deleted holiday will be tagged to Home, General.

Add New Location
Add a new location tag to the selected holiday.

View Locations
View list of location(s) tagged to holiday.

View Locations Map
Display a map view with location markers of locations that were tagged to the selected holiday. Zoom in or out to display markers depending on distance between locations. It may take a while for the map to load. Please note that the feature required data network connection.

View HOME Locations
View list of location(s) tagged to Home, General.
