MAIN MENU

Main screen provides quick access to main programs:

  • Locations
  • Vacations
  • Add New Location
  • Last Location/Default Location
  • More Option
  • Check List
  • Shopping List

Locations
Access to Locations list:

  • add, view, edit and delete location.
  • manage location photo album.
  • take photo and tag to location.
Go to Locations help

Vacations
Access to Vacations/Holidays list:

  • add, view, edit and delete holiday.
  • view vacation/holiday album.
  • view vacation locations list and map
Go to Vacations help

Add Location
The button on the left, below the Vacations menu, provides shortcut to add new location.

Last Location/Default Location
Displays the last location added or one selected by user via View Location.

Photo Album/Camera
Photo album for reviewing picture of the highlighted Location, the icon will not be displayed if no picture is available. Camera button for quick access to the device default camera application, picture taken will automatically be tagged to the location.

More Option
Holiday Checklist and Shopping list can be accessed via sliding menu, an orange button at bottom of screen. Slide the button upward to reveal menu. To close and return to previous screen, slide button downward to close screen.

Quick Conversion Screen A text box provides simple foreign currency conversion, display equivalent of home currency and a convenient tip computation, the tips amount is displayed in foreign currency. User need to set up foreign exchange and customary tips rates of visiting country in Setup Fx Rate under menu.

Check List
A vacation check list manages holiday To Do tasks and a packing list. A preloaded sample lists are provided. User can customizable the list and saved as user default.

Shopping List
A shopping list for both home and holiday shopping reminder, a picture can be tagged to item for quick reference.
