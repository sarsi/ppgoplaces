VIEW LOCATIONS

View list of locations. Click on any location to display the location address at the top of the list. If there are pictures available in the selected location, a location photo album button will display on the top right of the display.

Camera button for quick access to the device default camera application, picture taken will automatically be tagged to the location

Add Location button
Click button to add new location.

Location Context Menu
Long press on location item to access to the below:

  • Edit
  • Delete
  • Set As Default
  • Add Photo To Album
  • View Location Map
  • Share Location

Edit
Edit Location details.

Delete
Delete selected Location.

Set As Default
Selected location as default. Quick access to location in main menu by click on drawer button Default Location. User can clear default via menu, Clear Default. When no default is selected, the main menu drawer button will default to Last Location.

Add Photo To Album
Tag pictures to location from device default photo gallery. Pictures tagged to location or other locations will not appear in the gallery.

View Location Map
Display a map view with location marker of the selected location. It may take a while for the map to load. Please note that the feature required data network connection.

Share Location
Share the selected location as a file attachment via email with another user(s). Upon receipt of the email, the receipent can click on the file attachement which automatically launch PPGoPlaces application to received location information. Receipent select Update Location to add this new location to his/her locations list. Note: The receipent needs to have PPGoPlaces in order to receive location.
