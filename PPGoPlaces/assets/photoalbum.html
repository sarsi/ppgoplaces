PHOTO ALBUM
Photo album displays pictures of a location or locations of an holiday.

Checkbox button
Tap on the checkbox at the bottom left screen to select or unselect a picture. User can select multiple picture for batch actions such as Share or Remove

Picture Menu
Long press on a photo to access to the below:

  • Share
  • Remove From Album
  • Delete
  • Picture Details
  • Rename Picture

Share
Share allows the user to share picture, such as via email, facebook or cloud storage depending on the applications available in the device. This action is available for selected pictures under Menu,Share Selected

Remove From Album
Remove or untag picture from current album. Note: picture is not deleted except the location or holiday information of the picture. User can still access picture via gallery or other photo viewers applications. This action is available for selected pictures under Menu,Remove Selected

Delete
Delete picture permanently from device. Note: Deleted picture is not recoverable and not accessible to other application unlike Remove From Album function.

Picture Details
Display details of picture including location name, location address and picture name.

Rename Picture
Rename or enter name of picture. Picture name created here is only store in program and do not change name of picture stored on device.
