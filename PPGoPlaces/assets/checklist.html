CHECK LIST

The holiday checklist comprises of four categories: Pre-Holiday, At The Last Minute, While On Holiday and Post-Holiday, and a separate holiday Packing list.

Add New button
Click button to add new item to checklist.

Item Context Menu
Long press on checklist item to access item action menu below:

  • Edit Item
    • Edit details of item, e.g. item description or tagging to a different category.
  • Delete Item
    • Delete item from list.

Category Drop Down List
Click on list to display drop down list. Selecting a category item will return a filtered display of that category. Select Packing List to display packing list. Packing list is a separate list from checklist and will not display in Show All Tasks mode

Show/Hide Toggle Button
Hide selected or completed tasks.

Menu-Delete All Completed
Delete all checked items, i.e. items that were marked as completed, items from both checklist and packing list.

Menu-Save As Default List
Both checklist and packing list can be customized, after making changes to the list, select the option to make up a backup copy of the new default lists.

Menu-Reset To Default List
Restore the previously saved Default list, this action will overwrite current list with the the previously saved copy . if none was previously make it will restore to default list.

Reset CheckList
Reload holiday checklist with factory default. Note: the action will overwrite any current checklist and packing list
